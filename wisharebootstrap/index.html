Welcome to WiShare

- About -

At Wishare, our goal is to make the process of connecting to WiFi as simple as possible. We developed WiShare to fix an issue that many of us run into on a regular basis: You invite a friend over, and the first thing they ask you is, "What's the WiFi password?". With WiShare, you won't have to break out the piece of paper with the 35 digit long password, now you can just send your friend a simple invite and they can connect without a hitch.

- Documentation -

APP - WiShare CLIENT USER - Invited by the host user to join their network CLIENT DEVICE - the device belonging to the client user CONFIGURED NETWORK LIST - list of network configurations stored on the host device CONTACT - methods of communication for a person that is stored for easier access CONTACT LIST - May contain multiple contacts. Needs permission to be accessed CREDENTIALS - network name and password DATA LIMIT - A limit set by the host of how much data the client can use on the network DEVICE - A device with network access. Can be a client device, host device, or both. FIREBASE SERVER - The server forwards messages between the client and host devices GOOGLE SERVER - Creates a token for new users HOST DEVICE - the device belonging to the host user HOST USER - the user responsible for adding new network configurations. Invites client users onto their configured networks. INIVITATION - A single time invitation or permanent invitation NETWORK CONFIGURATION - connection information and credentials OPTIONS - Ability for host to set parameters for their invites to the client PASSWORD - the password of the network PERMANENT INVITATION - Host user gives client user unlimited access to the network POSTGRES DATABASE - Stores user’s tokens, phone numbers and other device information. PHONE NUMBER - Phone number of the device PROXIMITY - Relative distance of the client’s device to the network TIME LIMIT - A limit set by the host of how long the client can stay connected to the network TOKEN - A unique identifier used by the firebase server to identify a device SINGLE TIME INVITATION - Host user gives client user one time access to the network SQLITE DATABASE - A database specific for each device. Stores contact information, token information, and network configurations. USER - A client user or host user that uses WiShare WIFI/NETWORK - Wireless internet
